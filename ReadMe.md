    <h2>üìñ Description</h2>
    <p>
        This "Twitter Clone" is what it says it is...a <strong>TWITTER CLONE</strong>. It does the most basic functions a social media platform can: make accounts, like and unlike tweets, and create or post tweets. It is a basic and simple version of X or Twitter as it doesn't have image uploading, account algorithms, nor any other advanced features.
    </p>
    <p>
        I was unable to provide any of the aforementioned features because of the lack of time and sleep <em>(literally it is 12:47 AM)</em>, but I still managed to follow the minimal instructions needed for this project. Below are further details.
    </p>
    
    <h2>‚ú® Features</h2>
    <ul>
        <li>Creating an account / Logging in an account</li>
        <li>Tweeting / Creating worded posts</li>
        <li>Liking and Unliking posts from other accounts</li>
        <li>Viewing other people's accounts</li>

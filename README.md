## About Twitter Clone

This "Twitter Clone" is what it says it is...a **TWITTER CLONE**. It does the most basic functions a social media platform can: make accounts, like and unlike tweets, and create or post tweets. It is a basic and simple version of X or Twitter as it doesn't have image uploading, account algorithms, nor any other advanced features.

I was unable to provide any of the aforementioned features because of the lack of time and sleep _(literally it is 12:47 AM)_, but I still managed to follow the minimal instructions needed for this project. Below are further details.
